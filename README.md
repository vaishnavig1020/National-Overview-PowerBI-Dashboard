# 🇮🇳 MyScheme India Dashboard | Power BI

## Overview

An interactive Power BI dashboard built using data from the Government of India's MyScheme Portal.

The dashboard enables users to explore 4,680+ government schemes through multiple analytical perspectives including national overview, state-wise analysis, gender-based insights, subsidy comparison, and eligibility analysis.

---

## Project Objective

The objective of this project is to convert raw government scheme data into an interactive Business Intelligence dashboard that helps users identify:

- Scheme distribution across India
- State-wise performance
- Subsidy trends
- Gender-focused schemes
- Category distribution
- Eligibility patterns
